# vscode快捷键

## 重要的

`ALT` + `click` 选中行 这个最关键

`CTRl` + `shift` + `H` 全局替换

`F2` rename

`CTRl` + ` 打开cmd 

## 不是很重要

`ALT` + 箭头 上下移动

 `ALT` + `Shift`+箭头 复制一行在上下

`CTRL` + `R` 切换文件夹

`ALT` + `Z` 切换自动换行

`shift`+`Alt` +`I`

`CTRL` + `D` 选中相同的元素 只有两个 再按一次可以加1

`CTRL` + `shift` + `L` 上面的升级版本 全选

`CTRL `+`F2` 同上

`CTRL` + `Shift` +`Space `  查看函数

`F12` go to define 其实`CTRL` + `click` 也可以有一样的效果

`ALT`+`F12` 查看 但是不跳转



`CTRl` + `shift` + `c` 打开cmd

`CTRL` + `K` + `R` 打开文件管理器

`CTRL` + `K` + `V` 侧边预览
