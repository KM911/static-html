题外话

工具是为了提高效率 我并不觉得使用gui就一定 怎么这么样 使用命令行就 巴拉巴拉

如果你觉得使用某种方式更加适合自己 有助于提高自己的效率 这样就够了 没有必要花费时间和别人争论怎么做更好之类的

就算你对于一个工具不是非常了解 就比如这里的git 和 Github Desktop

掌握一些基本的技巧 就可以体验到新的工具 带来的好处 所以多看看 有什么新奇的玩意 然后学习一下吧
